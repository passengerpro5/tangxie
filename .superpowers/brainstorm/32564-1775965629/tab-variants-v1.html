Tab 切换 UI/UX 方案

这四套不是只改一级导航，而是给你一整套 tab 语言：主导航、页内切换、以及 `日 / 周 / 月 / 年` 这种模式切换会一起统一。你可以直接选 `A / B / C / D`，也可以说“主导航用 A，二级用 D”这种混搭。

A 胶囊分层

更温和、更稳的分层胶囊

一级导航是主胶囊条，二级和三级是更轻的胶囊层级。整体最温和，也最符合当前品牌方向。

B 下划线导航

更像真正的页面级导航

一级和二级都更偏页面导航范式，强调结构清晰。视觉更轻，但导航感更强。

一级 tab 被做成更有存在感的导航卡，二级和三级则保持轻量。适合让主切换显得很重要。

D 浮岛混合

主导航像浮岛，二三级更轻

一级导航做成悬浮式高权重模块，二级和三级则分别用圆角标签和小模式块区分，层次感最明显。
