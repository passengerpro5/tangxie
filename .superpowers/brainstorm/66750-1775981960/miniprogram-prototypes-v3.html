糖蟹小程序高保真原型 V3

这版已经把你确认过的规则合并进完整小程序方案：`浮岛混合` 作为统一 tab 语言，顶部尽量不跨行，`今日节奏` 只出现在 `日程` 首页，`事项 / 排期` 直接进入主内容，`任务详情` 只做信息展示升级。

首页仍是日程 日程 / 事项 / 排期功能不改 今日节奏只属于日程页 任务详情不新增操作

日程首页

首页保留最高层级的 `日程 / 事项 / 排期` 浮岛导航。二级模式和时间粒度压缩到同一控制带里，默认只有一行式 `今日节奏` 摘要，不抢主工作区高度。

审核重点

  • 首页是不是更轻了，但仍然像真正的日程首页。
  • tab 层级够高，但没有压得太重。
  • 摘要条够克制，时间轴仍然是视觉主角。

排期页

审核重点

  • 顶部是否足够短，但还保留了排期视图的专业感。
  • 规划主体是否更像计划，而不是冷硬图表。
  • 时间粒度切换是否清楚。
